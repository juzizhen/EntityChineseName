# 实体中文名 - Entity Chinese Name

 一个为 Minecraft 生物添加随机中文名的 Fabric 模组。
 
 ![Icon](https://raw.githubusercontent.com/juzizhen/EntityChineseName/main/src/main/resources/assets/entitychinesename/icon.png)

## 功能特点

- **随机中文名生成**：为游戏中新生成的生物实体随机赋予中文姓名
- **多样化姓名组合**：
  - 支持两字、三字、四字姓名的随机生成
  - 区分男性与女性姓名风格
  - 支持单姓和复姓组合
- **完全可配置**：
  - 所有概率均可通过配置界面调整
  - 可自定义姓氏和名字列表
  - 可选择为哪些类型的生物命名
- **智能排除**：
  - 默认排除末影龙（可配置）
  - 可自定义排除特定实体类型
- **简单易用**：
  - 完全兼容 ModMenu，配置界面友好
  - 支持多语言国际化

## 使用效果

当生物生成时，它们将自动获得一个随机的中文名字，例如：
- 男性名字：张勇、李伟、欧阳明辉
- 女性名字：王芳、赵秀英、上官婉婷

## 兼容性

- Minecraft 版本：1.20.1-1.20.6
- Mod 加载器：Fabric
- 依赖：Fabric API、Cloth Config、ModMenu

## 配置说明

所有配置均可通过游戏内的 ModMenu 界面进行调整：
- 姓名长度概率（两字、三字、四字）
- 性别概率（男性/女性）
- 姓氏类型概率（单姓/复姓）
- 实体类型过滤（被动/敌对/中立/BOSS）
- 自定义姓名列表

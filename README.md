A Fabric mod that adds random Chinese names to Minecraft mobs.

## Features

- **Random Chinese Name Generation:** Randomly assigns Chinese names to newly spawned mob entities in the game.
- **Diverse Name Combinations:**
- Supports random generation of two-character, three-character, and four-character names.
- Distinguishes between male and female name styles.
- Supports single and double surnames.
- **Fully Configurable:**
- All probabilities can be adjusted through the configuration interface.
- Customizable surname and given name lists.
- Select which types of mobs to name.
- **Intelligent Exclusion:**
- Excludes the Ender Dragon by default (configurable).
- Customizable exclusion of specific entity types.
- **Simple and Easy to Use:**
- Fully compatible with ModMenu, with a user-friendly configuration interface.
- Supports multi-language localization.

## Usage Example

When mobs spawn, they will automatically receive a random Chinese name, for example:
- Male names: Zhang Yong, Li Wei, Ouyang Minghui
- Female names: Wang Fang, Zhao Xiuying, Shangguan Wanting

## Compatibility

- Minecraft Version: 1.20.1-1.20.6
- Mod Loader: Fabric
- Dependencies: Fabric API, Cloth Config, ModMenu

## Configuration Instructions

All configurations can be adjusted through the in-game ModMenu interface:
- Name length probability (two-character, three-character, four-character)
- Gender probability (male/female)
- Surname type probability (single/double surname)
- Entity type filtering (passive/hostile/neutral/BOSS)
- Custom name lists

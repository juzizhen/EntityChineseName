# Entity Chinese Name

A Fabric mod that adds random Chinese names to Minecraft mobs.

## Features

- **Random Chinese Name Generator**: Randomly assigns Chinese names to newly spawned mobs in the game.

- **Diverse Name Combinations**:

- Supports random generation of two-, three-, and four-character names.

- Differentiates between male and female name styles.

- Supports single and compound surname combinations.

- **Fully Configurable**:

- All probabilities can be adjusted through the configuration interface.

- Customizable surname and given name lists.

- Selectable mob types to name.

- **Smart Exclusion**:

- Default exclusion of the Ender Dragon (configurable).

- Customizable exclusion of specific entity types.

- **Easy to Use**:

- Fully compatible with ModMenu, user-friendly configuration interface.

- Supports multilingual internationalization.

## Usage

When mobs spawn, they will automatically receive a random Chinese name, for example:

- Male names: Zhang Yong, Li Wei, Ouyang Minghui

- Female names: Wang Fang, Zhao Xiuying, Shangguan Wanting

## Compatibility

- Minecraft Version: 1.20.1

- Mod Loader: Fabric

- Dependencies: Fabric API, Cloth Config, ModMenu

## Configuration Instructions

All configurations can be adjusted through the in-game ModMenu interface:

- Name length probability (two characters, three characters, four characters)

- Gender probability (male/female)

- Surname type probability (single surname/compound surname)

- Entity type filtering (passive/hostile/neutral/BOSS)

- Custom name list
